Инструкция пользователя

Приложение предназначено для определения ретранслируемых радиочастот на станциях и оперативного доступа к данной информации.

Интерфейс максимально упрощен. В строке поиска необходимо начать вводить название станции, ниже отображается перечень станций с совпадением поиска. При клике на требуемую станцию отобразится информация о ней.

Для дополнительного отображения инструкции по настройке радиостанции необходимо во втором поле выбрать требуемую модель и кликнуть кнопку "Показать инструкцию". Будет показана инструкция с информацией по выбранной ранее частоте в верхней части экрана.
