О программе MHz

В основе создания приложения лежит идея максимального упрощения доступа к справочной информации о средствах связи и различных конфигурационных данных, необходимых сотрудникам локомотивной сферы в своей повседневной работе.

Приложение для OS Android, MHz, предназначено для корпоративного использования и содержит сведения по радиочастотам, используемым на железнодорожных станциях для закрепления подвижных составах.

Также в базе данных имеются инструкции по настройке радиочастот приминительно ко всем типам используемых на сети дорог радиостанциям.

Реализован алгоритм синхронизации данных который позволяет централизованно поддерживать всю локальную информацию в актуальном состоянии и как следствие для использования приложения не требуется подключение к сети интернет.

Проект развивается на некоммерческой основе, будующая функциональность и архитектурные идеи формируются на основе обратной связи от сотрудников, использующих приложение в своей работе. Команда разработки будет благодарна за конструктивную обратную связь с замечаниями и предложениями. Телеграм для связи @anbnH
